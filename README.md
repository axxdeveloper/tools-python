# tools-python
## splitfile.py
Split large file to smallers, every smaller file contains 100000 lines.<br /><br />
Ex. <br />
python splitfile.py D:\largefile.log<br />
largefile.log will be splitted to<br />
largefile.1.log<br />
largefile.2.log<br />
largefile.3.log<br />
largefile.4.log<br />
largefile.5.log<br />
...etc<br />
